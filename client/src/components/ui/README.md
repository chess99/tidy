本目录提供可复用的基础 UI 原语（按钮/输入/弹窗等），不承载业务逻辑。
输入：组件 props、样式类名；输出：一致的交互与视觉组件。
更新规则：新增/修改 UI 原语或依赖形状变化时，更新本 README + `client/src/components/README.md`。

### 文件

- `button.jsx`：按钮组件与变体样式。
- `input.jsx`：输入框组件。
- `select.jsx`：选择器组件。
- `checkbox.jsx`：复选框组件。
- `switch.jsx`：开关组件。
- `tabs.jsx`：Tabs 组件。
- `dialog.jsx`：对话框/模态框组件。
- `popover.jsx`：Popover 悬浮层组件。
- `tooltip.jsx`：提示气泡组件。
- `command.jsx`：命令面板/可搜索列表组件。
- `calendar.jsx`：日历组件（日期选择基础）。
- `date-range-picker.jsx`：日期范围选择组合组件。
- `separator.jsx`：分隔线组件。
- `badge.jsx`：徽标/标签组件。
