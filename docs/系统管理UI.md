# 系统管理 UI

系统管理是 Tidy 的“后台控制台”：**先配置扫描/并发，再触发任务并观察进度**，避免在多个页面之间来回切换。

## 入口

- 主界面顶部右侧：**系统管理** → 进入系统管理视图；**返回** → 回到“全部文件/文件夹归档”。

## 布局（immich 风格）

- **左列：配置**
  - 扫描源目录（启用/停用/移除）
  - 扫描类型（后缀）
  - 排除规则（glob）
  - 任务并发（仅此一处）
  - 人脸维护/同步对账等维护项
- **右列：任务队列**
  - 每类任务一张卡片，卡片顶部状态条直接展示：**正在处理 / 准备处理 / 空闲 / 上次失败**
  - 可直接触发（缺失/补扫、全部、开始）并支持取消/重试
