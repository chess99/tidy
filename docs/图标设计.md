# 图标设计（Tidy）

## 设计创意（为什么长这样）

### 核心语义

Tidy 的核心是“整理 / 归档 / 去重”：把混乱的库变成可管理的结构，并且有 AI 辅助（人脸 / CLIP）。
图标需要同时做到：

- 小尺寸可读（16–32px 仍能辨识）
- 传达“整理 / 分层 / 归类”的方向感
- 轻量提示“智能 / AI”但不喧宾夺主

### 形态

- **圆角方形容器**：桌面 / 移动 / 浏览器 tab 的通用范式，识别强。
- **三层堆叠卡片**：代表“分类后的内容层级 / 列表条目”，也暗合“去重后保留主副本”的结构。
- **三条强调色短条**：象征“被标记 / 被归类”的条目，增强小尺寸辨识度。
- **小星芒**：轻量表达 AI 辅助（“智能整理”），避免复杂元素导致缩小后糊成一团。

### 色彩（极简扁平）

- 深色底（接近 `#0b0b0f`）：与当前 UI 氛围一致。
- 卡片深灰（接近 `#161824`）：形成层次。
- 强调色青绿（接近 `#2dd4bf`）：科技感、克制、对比度高。

## 资产与入口（在哪里改）

### Web（favicon + tab 标题）

- 标题：`client/index.html` 的 `<title>`
- 图标：`client/public/icon.svg`（由 `client/index.html` 的 `<link rel="icon">` 引用）

### Desktop（Electron 打包图标）

- 图标源：同一个 `client/public/icon.svg`
- 生成：`desktop/scripts/generate-icons.mjs`（生成到 `desktop/assets/`）
- 使用：`desktop/package.json` 的 `build.mac.icon` / `build.win.icon`

> 注意：`desktop/assets/` 下的 `icon.*` 为生成产物，默认由根 `.gitignore` 忽略。

## AI 生成 prompt（可复现）

你可以用任意支持 SVG 输出的图像模型/矢量生成工具，核心约束是“小尺寸可读 + 极简扁平”。以下 prompt 可直接用于生成迭代。

### Prompt（中文）

生成一个 256x256 的 SVG 矢量图标（`viewBox="0 0 256 256"`），用于桌面 App 与浏览器 favicon。风格极简扁平、几何形状、无照片感、无复杂渐变，要求 16–32px 缩略仍清晰。

构图：
- 圆角方形底板
- 底板内有 3 条横向圆角矩形“卡片/列表项”堆叠
- 每条卡片左侧有一段高对比强调色短条（像标签）
- 右上角有一个很小的四角星芒（象征 AI 辅助）

颜色：
- 背景：接近 `#0b0b0f`
- 卡片：接近 `#161824`
- 强调色：接近 `#2dd4bf`
- 星芒：接近 `#e6e6ef`

输出要求：只输出 SVG（不要 PNG/JPG），不要文字，不要多余装饰。

### Prompt（English）

Create a minimal flat **SVG app icon** with `viewBox="0 0 256 256"` for an app called “Tidy”. The icon must stay crisp at 16–32px.

Composition:
- Rounded square background
- Three stacked rounded rectangles representing sorted list items/cards
- A short high-contrast accent bar on the left of each item
- A tiny four-point sparkle in the top-right corner to hint “AI assistance”

Style:
- Minimal flat geometric shapes, no photo realism
- No complex gradients, no text

Colors:
- Background near `#0b0b0f`
- Cards near `#161824`
- Accent near `#2dd4bf`
- Sparkle near `#e6e6ef`

Output: SVG only.
