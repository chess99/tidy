# 图标设计（Tidy）

## 设计创意（为什么长这样）

## 图标设计最佳实践（小尺寸可读）

面向桌面 App / 浏览器 favicon，最重要的不是“画得多漂亮”，而是**16–32px 仍可辨识**。业界常用准则：

- **单一主轮廓（single strong silhouette）**：最多 1 个主形状 + 1 个辅助点（比如小星芒），避免多物体叠加。
- **少元素、少层级**：小尺寸下细节必丢；优先用大面/粗线条表达含义。
- **高对比**：前景/背景的明度对比足够大；避免低对比渐变当作信息载体。
- **留安全边距（safe area）**：重要形状不要贴边，避免系统圆角裁切后变形。
- **可缩放一致性**：在 16/32/64/128/256 都预览一遍；必要时做“小尺寸特供版”（简化细节）。
- **颜色克制**：2–3 个主色足够；过多颜色会在缩小后“糊”。
- **避免文字**：文字在 16–32px 基本不可读，且不同系统渲染差异大。

### 核心语义

Tidy 的核心是“整理 / 归档 / 去重”：把混乱的库变成可管理的结构，并且有 AI 辅助（人脸 / CLIP）。
图标需要同时做到：

- 小尺寸可读（16–32px 仍能辨识）
- 传达“整理 / 分层 / 归类”的方向感
- 轻量提示“智能 / AI”但不喧宾夺主

### 形态

- **圆角方形容器**：桌面 / 移动 / 浏览器 tab 的通用范式，识别强。
- **三层堆叠卡片**：代表“分类后的内容层级 / 列表条目”，也暗合“去重后保留主副本”的结构。
- **三条强调色短条**：象征“被标记 / 被归类”的条目，增强小尺寸辨识度。
- **小星芒**：轻量表达 AI 辅助（“智能整理”），避免复杂元素导致缩小后糊成一团。

### 色彩（极简扁平）

- 深色底（接近 `#0b0b0f`）：与当前 UI 氛围一致。
- 卡片深灰（接近 `#161824`）：形成层次。
- 强调色青绿（接近 `#2dd4bf`）：科技感、克制、对比度高。

## 更换图标的流程（下次你自己跑）

目标：只维护一份“源 PNG”，同时一键同步 Web favicon + Desktop app icon。

1) 把你最新的图标 PNG 放到仓库根目录（任意文件名都可以），假设叫 `Generated Image 2.png`
2) 运行下面这些命令（macOS）：

```bash
cd /Users/zcs/code2/tidy

# 1) 归档源文件（assets 里只保留这一份留底）
mkdir -p docs/assets
cp "Generated Image 2.png" docs/assets/tidy-icon-source.png

# 2) Web / Desktop 的 canonical 源：1024×1024 icon.png
cp "Generated Image 2.png" client/public/icon.png

# 3) Web 小尺寸 favicon（避免网页加载 1024 大图）
sips -Z 32 client/public/icon.png --out client/public/favicon-32.png
sips -Z 16 client/public/icon.png --out client/public/favicon-16.png
sips -Z 180 client/public/icon.png --out client/public/apple-touch-icon.png

# 4) 生成 Desktop 的 icns/ico
cd desktop && npm run generate:icons

# 5) 重建 Web 产物（client/dist）
cd ../client && npm run build

# 6) 重打 mac 安装包（可选：如果要发布）
cd ../desktop && npm run dist:mac
```

3) 清理：把仓库根目录里临时的 PNG 删除掉（避免散落）

> 说明：桌面端会从 `client/public/icon.png` 生成 `desktop/assets/icon.icns`/`icon.ico`；`desktop/assets/icon.*` 是生成物，会被 `.gitignore` 忽略。

### Web（favicon + tab 标题）

- 标题：`client/index.html` 的 `<title>`
- 运行时小图标：`client/public/favicon-16.png`、`client/public/favicon-32.png`（由 `client/index.html` 引用）
- Apple touch：`client/public/apple-touch-icon.png`
- 图标源（用于生成桌面 icns/ico）：`client/public/icon.png`（建议 1024×1024，透明底）

### Desktop（Electron 打包图标）

- 图标源：优先使用 `client/public/icon.png`（若不存在则回退到 `client/public/icon.svg`）
- 生成：`desktop/scripts/generate-icons.mjs`（生成到 `desktop/assets/`）
- 使用：`desktop/package.json` 的 `build.mac.icon` / `build.win.icon`

> 注意：`desktop/assets/` 下的 `icon.*` 为生成产物，默认由根 `.gitignore` 忽略。

## AI 生成 prompt（可复现）

很多平台（如 Midjourney、部分“文生图”工具）**不支持直接输出 SVG**。推荐流程是：

1) 先生成 **1024×1024 PNG（透明底）**，确保图形极简、边界清晰、颜色可控  
2) 再用矢量化工具转成 SVG（Inkscape / Illustrator Image Trace / 其它矢量化工具）  
3) 用生成的 SVG 替换 `client/public/icon.svg`（再由 `desktop/scripts/generate-icons.mjs` 生成 icns/ico）

### Prompt（中文，通用文生图）

生成一个 1024×1024 的应用图标 PNG，透明背景（或纯色背景也可，但优先透明），用于名为 “Tidy” 的桌面应用。风格：极简扁平、几何形状、无照片感、无 3D、无复杂渐变。要求 16–32px 缩略仍清晰：请保证**只有一个主轮廓**，细节极少，不要细线条。

构图：
- 圆角方形底板（深色）
- 底板内是“堆叠”的单一符号（例如 2–3 层叠片/叠卡），线条粗、层次清晰
- 右上角一个很小的四角星芒（象征 AI 辅助），不要额外元素

颜色（可直接指定接近值）：
- 背景：`#0b0b0f`
- 卡片：`#161824`
- 强调色：`#2dd4bf`
- 星芒：`#e6e6ef`

负面约束：
- 不要文字、不要摄影、不要写实材质、不要复杂阴影、不要多余图案、不要网络节点/细碎装饰

### Prompt（Midjourney 示例）

`minimal flat app icon, rounded square background, three stacked rounded rectangles like sorted list items, small teal accent bars on the left of each row, tiny sparkle in the top-right, high contrast, clean geometric shapes, no text, no 3d, no photo, transparent background --ar 1:1 --style raw`

### Prompt（English, generic image model）

Generate a 1024×1024 PNG app icon for “Tidy”, preferably with a transparent background. Style: minimal flat geometric, high contrast, readable at 16–32px, no photo realism, no 3D, no complex gradients, no text.

Composition:
- Rounded square dark background
- Three stacked rounded rectangles (sorted list items/cards)
- Short teal accent bar on the left of each item
- Tiny four-point sparkle at the top-right (AI hint)

Palette:
- Background near #0b0b0f
- Cards near #161824
- Accent near #2dd4bf
- Sparkle near #e6e6ef

Output: PNG.

## 新版图标设计方案（2026-01）

### 设计理念

**核心概念**：**"智能整理，化繁为简"** - 将混乱的文件通过 AI 辅助变成有序的归档

基于对 Tidy 项目的深入分析，新设计理念强调：

#### 功能特色
- **主要功能**：照片/视频去重、智能分类、AI 辅助
- **核心价值**：从混乱到有序、内容感知、智能化
- **技术特色**：人脸识别、CLIP 搜索、内容哈希去重

#### 视觉语言
- **形状**：圆润现代，体现友好易用
- **结构**：层次清晰，体现整理归档
- **色彩**：科技感配色，突出 AI 智能
- **风格**：简约现代，适配多平台

### 四种设计方案

#### 方案一：智能文件夹（推荐）

**概念**：直观的文件整理视觉，突出核心功能

```
Create a modern app icon for a photo organizing tool called "Tidy". Design a clean, rounded square icon with:

MAIN ELEMENT: A stylized folder or container in gradient blue (#2563eb to #1d4ed8)
ORGANIZING VISUAL: Three small photo thumbnails or cards floating above/emerging from the folder, arranged in a neat stack or grid
AI INDICATOR: Subtle sparkles or a small neural network pattern around the photos
BACKGROUND: Clean white or very light gray (#f8fafc)
STYLE: Modern, minimalist, iOS-style with subtle shadows and rounded corners
SIZE: 1024x1024 pixels, transparent background
COLORS: Blue gradients, white, soft gray accents, tiny golden sparkles

The icon should convey "smart photo organization" and "AI-powered file management" in a clean, professional way.
```

**优势**：直观易懂，符合用户认知，专业感强

#### 方案二：智能魔法棒

**概念**：强调 AI 的神奇整理能力

```
Design a sleek app icon for "Tidy" - an AI-powered photo organizer. Create:

MAIN ELEMENT: A modern magic wand or organizing tool with gradient purple-to-blue handle (#8b5cf6 to #3b82f6)
TRANSFORMATION VISUAL: Small photo icons transforming from scattered/messy on left to organized grid on right
AI SPARKLES: Bright particles or stars emanating from the wand tip
BACKGROUND: Clean rounded square container in dark navy (#1e293b)
STYLE: Modern 3D-ish with subtle depth, clean lines
SIZE: 1024x1024, transparent background
COLORS: Purple-blue gradients, white/light blue accents, golden sparkles

Should feel magical but professional, conveying "instant organization" and "AI magic".
```

**优势**：突出 AI 能力，有趣生动，强调智能化特色

#### 方案三：智能眼睛

**概念**：体现 AI 视觉识别和内容感知能力

```
Create an app icon for "Tidy" photo organizer featuring:

MAIN ELEMENT: A stylized eye or lens shape in teal gradient (#14b8a6 to #0891b2)
VISION CONCEPT: The pupil shows a grid of organized photos or thumbnails
AI NEURAL: Subtle circuit patterns or neural connections around the eye
CONTAINER: Rounded square background in soft dark gray (#374151)
DEPTH: Modern flat design with subtle gradients and highlights
SIZE: 1024x1024 pixels, transparent background
COLORS: Teal/cyan gradients, dark gray, white highlights, soft blue accents

The icon should represent "AI vision", "smart recognition", and "seeing order in chaos".
```

**优势**：体现人脸识别和 CLIP 搜索功能，突出 AI 视觉能力

#### 方案四：立体方块组合

**概念**：渐进式整理，现代科技感

```
Design a modern 3D-style icon for "Tidy" AI photo organizer:

MAIN ELEMENT: Three floating cubes/blocks in different sizes, arranged in ascending order
CUBE STYLING: Each cube shows a different photo/image on its face, with rounded edges
COLOR SCHEME: Gradient from coral (#f97316) to pink (#ec4899) to purple (#a855f7)
AI ELEMENTS: Small glowing particles connecting the cubes, showing AI processing
BACKGROUND: Clean white or transparent
LIGHTING: Soft shadows beneath, gentle highlights on top edges
SIZE: 1024x1024, transparent background
STYLE: Modern 3D isometric, clean and professional

Should convey "building organization", "smart sorting", and "progressive improvement".
```

**优势**：现代 3D 感，层次丰富，年轻化、科技感强

### 方案对比与推荐

| 方案 | 核心概念 | 优势 | 适用场景 |
|------|----------|------|----------|
| **智能文件夹** | 整理归档 | 直观易懂，符合用户认知 | 通用推荐，专业感强 |
| **智能魔法棒** | AI 魔法 | 突出 AI 能力，有趣生动 | 强调智能化特色 |
| **智能眼睛** | AI 视觉 | 体现人脸识别和 CLIP 搜索 | 突出 AI 视觉能力 |
| **立体方块** | 渐进整理 | 现代3D感，层次丰富 | 年轻化、科技感 |

### 推荐使用顺序

1. **首选：智能文件夹** - 最符合产品定位，专业且易理解
2. **备选：智能眼睛** - 突出 AI 视觉特色，与人脸识别/CLIP 功能呼应
3. **创意：智能魔法棒** - 强调 AI 的神奇效果
4. **现代：立体方块** - 3D 风格，适合追求视觉冲击力

### 实施流程

1. **生成图片**：选择一个 prompt，在 AI 绘图工具中生成
2. **格式要求**：1024x1024 PNG，透明背景
3. **矢量化**：使用 Adobe Illustrator 或 Inkscape 转换为 SVG
4. **替换文件**：更新 `client/public/icon.svg`
5. **生成图标**：运行 `npm run desktop:dist` 自动生成各平台图标

### 设计原则

- **可扩展性**：确保在 16-32px 小尺寸下仍清晰可辨
- **品牌一致性**：与 Tidy 的现代、专业、智能定位保持一致
- **跨平台适配**：适用于 Web favicon、macOS、Windows 等多平台
- **功能表达**：图标应能直观传达"智能文件整理"的核心价值
