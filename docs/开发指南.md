# 开发指南

## 运行环境

- Node.js：建议 20+（前端 Vite 对 Node 版本有要求）

## 本地启动

1. 安装依赖

```bash
cd server && npm install
cd ../client && npm install
```

1. 启动服务端

```bash
cd server
npm run dev
```

1. 启动前端

```bash
cd client
npm run dev
```

## 人脸（可选功能）

### 设计原则

- **模型文件不进 git**：模型属于大体积二进制资源；仓库只保留脚本，开发者本地初始化到 `server/models/`。

### 初始化步骤

```bash
cd server
npm install
npm run models:setup
```

### 网络/代理提示（tfjs-node 安装失败）

```bash
cd server
HTTPS_PROXY=http://127.0.0.1:7897 npm i @tensorflow/tfjs-node
```

### InsightFace（可选增强）

如果你希望更接近 Immich/PhotoPrism 的人像体验（更强检测/embedding/可做 GPU 加速），可以使用独立的 Python 人脸服务：`face-service/`。

- 说明与启动方式见：`face-service/README.md`
